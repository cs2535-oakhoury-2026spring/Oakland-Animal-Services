# ARD-002: Pet Data Normalization

## Context

Because rescue groups do not allow custom fields in their entry, they had to put information in section that didn't correspond to the information. For example, they would put the pet's handling color as `otherNames`. Which cause confusion for us initially.

## Decision

For the time being we did not solve this and only left a comment next to the field that is being misused. In the future we plan to add a normalization step where we would extract the information from the misused fields and put them in the correct fields. This would allow us to have a more consistent data structure.
