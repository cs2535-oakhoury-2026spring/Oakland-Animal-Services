# Oakland Animal Services Portal - User Documentation

Welcome to the Oakland Animal Services Portal user documentation. This guide provides comprehensive information about using the portal from a user's perspective.

**For veterinary and behavioral staff** with full system access.

## Legal and Privacy

**Data Handling**: All animal data and user observations are stored securely and accessible only to authorized personnel with appropriate role-based permissions.

**User Privacy**: User activity is logged for security and auditing purposes. Logs are accessible only to administrators and used solely for system integrity and compliance.

**Account Security**: Users are responsible for maintaining the confidentiality of their login credentials. Report any suspected compromise immediately to administrators.
