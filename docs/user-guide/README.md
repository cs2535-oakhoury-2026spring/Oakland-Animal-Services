# Oakland Animal Services Portal -- User Documentation

This folder contains user-facing documentation for the Oakland Animal Services Portal.

## Contents

- **[USER_GUIDE.md](./USER_GUIDE.md)** -- Complete user guide covering all roles and features

The user guide includes:

1. **Executive Summary** -- Problem, solution, and key capabilities
2. **Getting Started** -- Login, navigation, QR scanning, dark mode
3. **Animal Detail Page** -- Summary tab, medical observations, behavior notes, AI summaries
4. **Volunteer Guide** -- Permissions, creating notes, tips for good observations
5. **Staff Guide** -- Editing/deleting notes, status tracking, activity log, limited user management
6. **Admin Guide** -- Full user management, activity log with auth events, location management, security
7. **Access Level Summary** -- Feature matrix by role
